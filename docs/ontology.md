# Ontology

The demo ontology is stored in [data/mini_ontology.json](../data/mini_ontology.json).

It includes:

- service catalog entries
- policy entries
- epistemic control keyword groups
- tool routing rules
- explicit concept definitions
- explicit relation edges
- adversarial block patterns

The validator exposes:

- concept extraction
- ontology hit lists
- lexical coverage
- node hit ratio
- relation validity ratio
- structured rejection reasons

This is not a large production ontology. It is a bounded test substrate used to make runtime validation behavior inspectable.
